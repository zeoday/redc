# GUI 设计规范

## 概述

本文档记录 RedC GUI 的设计规范，包括配色、边框、按钮等样式定义，供后续维护参考。

---

## 1. 页面整体风格

### 1.1 卡片容器

```html
<div class="bg-white rounded-xl border border-gray-100 p-5">
```

| 属性 | 值 |
|------|-----|
| 背景色 | `bg-white` (白色) |
| 圆角 | `rounded-xl` (12px) |
| 边框 | `border border-gray-100` (灰色-100, 1px) |
| 内边距 | `p-5` (20px) 或 `p-4` (16px) |

### 1.2 页面背景

```html
<div class="min-h-screen bg-gray-50">
```

| 属性 | 值 |
|------|-----|
| 最小高度 | `min-h-screen` (100vh) |
| 背景色 | `bg-gray-50` (灰色-50) |

## 3. 按钮规范

### 3.1 主要按钮（Primary / 蓝色）

```html
<button class="h-10 px-5 bg-blue-600 text-white text-[13px] font-medium rounded-lg hover:bg-blue-700 transition-colors cursor-pointer">
```

| 属性 | 值 |
|------|-----|
| 高度 | `h-10` (40px) |
| 水平内边距 | `px-5` (20px) |
| 背景色 | `bg-blue-600` |
| 文字颜色 | `text-white` |
| 文字大小 | `text-[13px]` |
| 字体粗细 | `font-medium` (500) |
| 圆角 | `rounded-lg` (8px) |
| 悬停背景 | `hover:bg-blue-700` |
| 过渡动画 | `transition-colors` |
| 光标 | `cursor-pointer`（Wails 桌面应用必需） |

### 3.2 成功按钮（Success / 绿色）


| 用途 | 颜色类 | 说明 |
|------|--------|------|
| 背景 | `bg-emerald-500` | 默认绿色 |
| 悬停 | `hover:bg-emerald-600` | 悬停加深 |

### 3.3 危险按钮（Danger / 红色）


| 用途 | 颜色类 | 说明 |
|------|--------|------|
| 背景 | `bg-red-500` | 默认红色 |


| 属性 | 值 |
|------|-----|
| 高度 | `h-10` (40px) |
| 水平内边距 | `px-5` (20px) |
| 背景色 | `bg-white` |
| 边框 | `border border-gray-300` |
| 文字颜色 | `text-gray-700` |
| 文字大小 | `text-[13px]` |
| 字体粗细 | `font-medium` |
| 圆角 | `rounded-lg` |
| 悬停背景 | `hover:bg-gray-50` |

### 3.5 幽灵按钮（Ghost / 边框）

```html

```

| 用途 | 颜色类 | 说明 |
|------|--------|------|
| 背景 | `bg-white` | 白色或透明 |
| 边框 | `border border-red-500` | 红色边框 |
| 文字 | `text-red-500` | 红色文字 |
| 悬停背景 | `hover:bg-red-50` | 浅红背景 |

### 3.6 小型按钮

```html
<button class="h-7 px-2.5 text-[10px] font-medium rounded-lg cursor-pointer">
```

| 属性 | 值 |
|------|-----|
| 高度 | `h-7` (28px) |
| 水平内边距 | `px-2.5` (10px) |
| 文字大小 | `text-[10px]` |

### 3.7 禁用状态

```html
<button class="disabled:opacity-50 disabled:cursor-not-allowed">
```

| 属性 | 值 |
|------|-----|
| 透明度 | `disabled:opacity-50` (50%) |
| 光标 | `disabled:cursor-not-allowed` |

> **重要**：RedC 是 Wails 桌面应用，不是浏览器环境。`<button>` 元素默认不显示手形光标，因此所有按钮必须显式添加 `cursor-pointer`。

## 4. 输入框规范

### 4.1 文本输入框（无边框）

```html
<input class="w-full h-10 px-3 text-[13px] bg-gray-50 border-0 rounded-lg text-gray-900 placeholder-gray-400 focus:ring-2 focus:ring-gray-900 focus:ring-offset-1 transition-shadow">
```

| 属性 | 值 |
|------|-----|
| 宽度 | `w-full` (100%) |
| 高度 | `h-10` (40px) |
| 内边距 | `px-3` (12px) |
| 背景色 | `bg-gray-50` |
| 边框 | `border-0` (无) |
| 圆角 | `rounded-lg` |
| 文字大小 | `text-[13px]` |
| 文字颜色 | `text-gray-900` |
| 占位符颜色 | `placeholder-gray-400` |
| 聚焦边框 | `focus:ring-2 focus:ring-gray-900 focus:ring-offset-1` |
| 过渡阴影 | `transition-shadow` |

### 4.2 文本输入框（带边框）

```html
<input class="w-full px-3 py-2 text-[13px] border border-gray-300 rounded-lg focus:outline-none focus:ring-2 focus:ring-gray-900">
```

| 属性 | 值 |
|------|-----|
| 边框 | `border border-gray-300` |
| 聚焦边框 | `focus:ring-2 focus:ring-gray-900` |

### 4.3 下拉选择框

```html
<select class="h-10 px-3 text-[13px] bg-gray-50 border-0 rounded-lg text-gray-900 focus:ring-2 focus:ring-gray-900">
```

### 4.4 聚焦样式规则

不同页面使用不同的聚焦色以匹配主色调：

| 页面类型 | 聚焦样式 | 使用场景 |
|----------|----------|----------|
| 通用页面（默认） | `focus:ring-2 focus:ring-gray-900` | Dashboard、模板、自定义部署等 |
| SSH 终端页面 | `focus:ring-2 focus:ring-red-500` | SSH 面板内输入框 |
| 蓝色语义输入框 | `focus:ring-2 focus:ring-blue-500 focus:border-blue-500` | 搜索框、Compose 配置项、特殊模块 |
| 轻量内联输入框 | `focus:outline-none focus:border-blue-400` | Settings Agent 配置、分页跳转 |

> 原则：聚焦色跟随页面主色调。通用页面用 `gray-900`，SSH 用 `red-500`，含搜索/筛选功能的嵌入式输入框用 `blue-500`。

## 10. 窗口自适应

> RedC 是 Wails 桌面应用，不存在移动端。以下断点用于窗口缩放时的自适应布局。

| 断点 | 前缀 | 窗口宽度 | 说明 |
|------|------|----------|------|
| 小窗口 | `sm:` | ≥640px | 窗口缩小时的紧凑布局 |
| 中窗口 | `md:` | ≥768px | 中等窗口布局 |
| 大窗口 | `lg:` | ≥1024px | 全屏/大窗口布局 |

**常用自适应写法：**

```html
<!-- 高度响应式 -->
<input class="h-9 sm:h-10 ...">

<!-- 文字大小响应式 -->
<div class="text-[12px] sm:text-[13px] ...">

<!-- 内边距响应式 -->
<button class="px-4 sm:px-5 ...">
```

## 15. Loading / Spinner 规范

### 15.1 通用旋转加载图标

```html

  <circle class="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" stroke-width="4"/>
```

| 场景 | 图标颜色 | 尺寸 |
|------|----------|------|

### 15.2 按钮加载状态

按钮在 loading 时应：
- 显示 spinner + 文字（如"部署中..."）
- 添加 `disabled` 属性防止重复点击
- 使用 `disabled:opacity-50` 降低透明度

```html
<button class="... disabled:opacity-50" disabled={loading}>
  {#if loading}
    <svg class="w-3.5 h-3.5 animate-spin" ...></svg>
    处理中...
  {:else}
    确认
  {/if}
</button>
```

## 18. 关键原则

1. **一致性优先**: 所有按钮高度统一使用 `h-10`，圆角统一使用 `rounded-lg`，所有可点击元素必须添加 `cursor-pointer`
2. **色彩语义**: 通用页面蓝色用于主要操作，SSH/AI Chat 页面红色用于主要操作，绿色用于成功状态，红色用于错误状态，amber 用于警告/执行中状态
3. **层级分明**: 页面背景(`bg-gray-50`) → 卡片背景(`bg-white`，对比突出) → 输入框背景(`bg-gray-50`，嵌套区分)
4. **窗口自适应**: 基于 Tailwind 断点实现窗口缩放自适应，小窗口用较小尺寸，大窗口自动适配
5. **可访问性**: 确保文字与背景有足够对比度，使用 `aria-label` 等属性
